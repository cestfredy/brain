# Commencer avec Microsoft Foundry

Ici, l'idée sera de découvrir comment microsoft nous permet de créer l'IA avec la techno la plus récente, sécure et à grande échelle.

## Qu'est-ce qu'une application IA ?

Charges de travail d'IA clés :
- IA génératice
- Agents et automatisation
- Speech
- Analyse de texte
- Vision par ordinateur
- Extraction d'informations

ML : permettre aux machines d'apprendre sans être explicitement programmées à le faire.

Types d'apprentissage automatique :
- Apprentissage supervisé et non supervisé :
    * regression pour prédire les prix
    * classification pour détecter si un courrier est indésirable 
    * clustering pour la segmentation des clients (non supervisé)

- Deep Learning : branche du ML utilisant les RNs avc plusieurs couches pour des taches de reconnaissances d'images et la synthèse vocale.

- IA Générative : utilise les fonctionnalités d'apprentissage profond pour créer du contenu (texte, images, audio, code) au lieu de classer pour de prédire les rsultats.
